# Portail Facturation

**Francais** | [English](#english)

---

## Francais

Portail Facturation est l'application E-D pour gerer les clients, devis, factures, recettes, produits et exports de facturation electronique.

### Fonctionnalites

- Gestion des clients, produits, devis et factures
- Suivi des recettes et des depenses
- Export PDF des documents commerciaux
- Generation de fichiers Factur-X / XML conformes EN 16931
- Synchronisation avec le compte e-d
- Interface disponible sur bureau et Android

### Plateformes

| Plateforme | Format |
|------------|--------|
| Windows | `.exe` (installeur NSIS) |
| Linux | `.tar.gz` |
| macOS | `.dmg` |
| Android | `.apk` |

### Installation

Telecharge la derniere version depuis les [releases GitHub](https://github.com/3yezz/pe-download/releases).

### Stack technique

- **Interface** : React
- **Bureau** : Electron
- **Mobile** : Android natif (WebView)
- **Build** : electron-builder, Gradle

## English

Portail Facturation is the E-D app for managing customers, quotes, invoices, revenue, products and electronic invoicing exports.

### Features

- Customer, product, quote and invoice management
- Revenue and expense tracking
- PDF export for commercial documents
- Factur-X / EN 16931 XML generation
- e-d account sync
- Desktop and Android interface

### Platforms

| Platform | Format |
|----------|--------|
| Windows | `.exe` (NSIS installer) |
| Linux | `.tar.gz` |
| macOS | `.dmg` |
| Android | `.apk` |

### Installation

Download the latest version from the [GitHub releases](https://github.com/3yezz/pe-download/releases).

### Tech stack

- **UI** : React
- **Desktop** : Electron
- **Mobile** : Android native (WebView)
- **Build** : electron-builder, Gradle
